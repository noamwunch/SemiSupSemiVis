# SemiSupSemiVis
Method for tagging semi-visible dark jets in a semi-supervised fashion.

# Use
1. Single run
        ```bash
        git clone https://github.com/noamwunch/SemiSupSemiVis
        cd SemiSupSemiVis
        . ./RUN/run_from_argseed.sh
        ```
2. Grid run on SGE cluster
        ```
        . ./RUN/CLUSTER/run_grid.sh
        ```
